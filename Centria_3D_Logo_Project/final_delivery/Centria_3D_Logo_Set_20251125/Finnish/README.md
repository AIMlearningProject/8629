# Finnish Logo Variants

## Contents

### STL Files (Ready for 3D Printing)
- `pin.stl` - Pin variant (3.5mm thickness)
- `magnet.stl` - Magnet variant (4.5mm thickness)
- `keyring.stl` - Keyring variant (5.5mm thickness)
- `cake_mould.stl` - Cake mould variant (10mm height)

### Blender Files (Source)
- Contains editable `.blend` files for modifications

### Renders (Previews)
- Preview images of each variant
- Use for reference and documentation

## Printing Instructions

See main `PRINTING_GUIDE.md` in root folder.

## Specifications

| Variant | Thickness | Special Features |
|---------|-----------|------------------|
| Pin | 3.5mm | Pin mount on back |
| Magnet | 4.5mm | 12mm magnet recess |
| Keyring | 5.5mm | Reinforced loop |
| Cake Mould | 10mm | Inverted relief |

**Material:** PLA or PETG recommended
**Infill:** 20-50% depending on variant
**Supports:** Not required
